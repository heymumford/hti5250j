  - pass-through (direct device passthrough)
  - transparent (intelligent routing based on format)

DIMENSION 2: Data Format (3 values)
  - text (plain text content)
  - scs-commands (Spooled Command Stream with control codes)
  - binary (raw binary with escape sequences)

DIMENSION 3: Session Type (3 values)
  - display (display-oriented session, screen output)
  - printer (printer-dedicated session)
  - dual (combined display and printer capabilities)

DIMENSION 4: Buffer Handling (3 values)
  - immediate (flush immediately without buffering)
  - buffered (collect data in memory buffer)
  - spooled (spool to queue for batch processing)

DIMENSION 5: Error Recovery (3 values)
  - retry (automatic retry on transient failure)
  - skip (skip failed items and continue)
  - abort (terminate operation on critical failure)
